# 🌳 ToStory - Interactive Digital Storytelling

[![PWA](https://img.shields.io/badge/PWA-5A0FC8?style=for-the-badge&logo=pwa&logoColor=white)](https://web.dev/progressive-web-apps/)

**ToStory** adalah sebuah platform web interaktif yang bertujuan untuk melestarikan dan memperkenalkan kekayaan cerita rakyat serta budaya lokal Indonesia melalui pendekatan digital yang modern. Versi pertama ini mengangkat keindahan dan sejarah dari **Desa Kakaskasen 2, Tomohon, Sulawesi Utara**.

## ✨ Fitur Utama

- **📖 Storytelling Interaktif**: Membaca cerita rakyat dengan ilustrasi yang dikurasi khusus dan transisi halaman yang mulus.
- **🎧 Immersive Atmosfer**: Dilengkapi dengan musik latar (BGM) untuk meningkatkan pengalaman imersif saat membaca.
- **🎮 Gamifikasi & Progres**: Sistem kuis interaktif dengan reward konfeti dan penyimpanan skor tertinggi menggunakan LocalStorage.
- **📱 Mobile-First & Responsive**: Desain skueomorfik premium yang dioptimalkan untuk perangkat mobile.
- **🖼️ Image Lightbox**: Jelajahi detail keindahan desa melalui galeri foto yang bisa diperbesar (*fullscreen*).
- **⚡ Progressive Web App (PWA)**: Dapat diinstal di layar beranda HP Anda dan diakses layaknya aplikasi asli.

## 🚀 Teknologi

- **Frontend**: React 18, TypeScript, Vite
- **Animasi**: Framer Motion (Route transitions, micro-interactions)
- **Styling**: Vanilla CSS (Premium design system, Glassmorphism)
- **PWA**: `vite-plugin-pwa`
- **Audio**: Web Audio API integration

## 🛠️ Instalasi Lokal

1.  **Clone Repositori**:
    ```bash
    git clone https://github.com/ShinZeleo/ToStory.git
    ```
2.  **Masuk ke Direktori**:
    ```bash
    cd ToStory
    ```
3.  **Instal Dependensi**:
    ```bash
    npm install
    ```
4.  **Jalankan Mode Pengembangan**:
    ```bash
    npm run dev
    ```

## 📦 Build untuk Produksi

Untuk menghasilkan file siap deploy (folder `dist`):
```bash
npm run build
```
